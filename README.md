#灵梦(reimu)
基于Live2d技术的虚拟形象物联网交互软件。

* 浙江大学2015届优秀毕业论文（设计）。

整个开发周期历时1年有余，之前因为写的丑所以不敢放上github，经过慢慢完善，现在从Coding.net移植到github上来，供有兴趣的人分享。

项目内容（即毕业设计报告）详见
